# TerraFarming テスト版

TerraFarming は、野菜を育てながら小惑星を少しずつ開拓していく、ほのぼの放置系ファーミングゲームです。

`index.html` をブラウザで開くと、現在のWeb試作を遊べます。

## テスト版の内容

- 小さな小惑星の畑に種を植える。
- 作物が時間経過で成長する。
- 収穫した野菜を倉庫に保存する。
- 植える作物を選んでから畑に植える。
- 作物ごとに酸素、水分、窒素の増え方が違う。
- 収穫によって食料、酸素、水分、窒素、テラポイントが増える。
- 倉庫の野菜を売るとコインを得られ、食料在庫は減る。
- 収穫した野菜2つを交配して、より強い種を作る。
- 作物のステータスに応じて、交配後の名前や色が変わる。
- 収穫を重ねると小惑星の環境値が改善する。
- 酸素、水分、窒素の条件を満たすと開拓段階が進む。
- 次の開拓段階まで、酸素、水分、窒素があといくつ必要か確認できる。
- 開拓段階に応じて、成長速度、売却価格、突然変異率にボーナスが付く。
- 開拓ミッションを達成したあと、自分で報告して報酬や解放要素を得る。
- コインで設備をアップグレードし、成長速度、売却価格、突然変異率を上げる。
- 売却は確認表示をはさんで確定する。
- 収穫やボタン操作に簡易効果音が鳴る。
