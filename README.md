# TerraFarming

野菜を育てながら小惑星を少しずつ開拓していく、縦型スマホ向けのほのぼの放置系ファーミングゲーム試作です。

## 公開設定

Netlify の公開対象は以下です。

```text
outputs/TerraFarming
```

`netlify.toml` に同じ設定を入れているため、Netlify 側では通常、ビルドコマンドなしで公開できます。

```text
Build command: 空欄
Publish directory: outputs/TerraFarming
```
